 🔐 Auth + Calculator App

A full-stack application featuring Google authentication and a backend-powered calculator. Users can securely log in, perform calculations, and view their previous history.

---

 🎯 Objective

To build a secure and interactive application with authentication and backend integration.

---

 ✨ Features

- 🔐 Google Authentication (Login)  
- 🧮 Perform addition & subtraction  
- 🔄 Backend API integration  
- 📜 Store and display calculation history  
- ⚡ Real-time result display  

---

 🔄 Project Flow

1. User logs in using Google  
2. Enters numbers on frontend  
3. Request sent to backend  
4. Backend performs calculation  
5. Result sent back and displayed  
6. History is stored and shown  

---

 🧠 Concepts Used

- API integration (Frontend ↔ Backend)  
- Authentication using Google API  
- CRUD operations (history)  
- DOM manipulation  
- State handling  

---

 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Express  
- API: Google Auth API
